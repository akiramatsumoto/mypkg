ロボットシステム学の講義で提出するためのROS 2のパッケージです.  
パソコンの電池残量をトピック```\batterylevel```から出します.
## ノード
### battery_monitor
パソコンの電池残量を調べ, 2秒ごとに送信します.
#### 送信先トピック
- batterylevel
## ROSバージョン
- ROS 2 Humble Hawksbill(自分のパソコン, GithubActionsどちらも)
